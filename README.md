Tugas UAS Program Grafika Komputer: 
Output program dan laporan berupa PDF yang berisi penjelasan kode program (boleh disertai komentar pada kode).
Disertakan juga video demo program berdurasi sekitar lima menit yang memperlihatkan proses menjalankan program.
Program harus menampilkan representasi objek menggunakan teori graf, serta menunjukkan bahwa objek tersebut dapat ditransformasikan (misalnya skala) dengan memanfaatkan matriks transformasi.
Buatlah program yang membangun objek atau gambar (berupa titik dan garis) yang disimpan dalam bentuk vertex, kemudian di-load kembali untuk ditampilkan sehingga terlihat bentuk objek yang dihasilkan.
